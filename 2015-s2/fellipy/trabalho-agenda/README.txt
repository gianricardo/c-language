#Programa

1-Abra um terminal na pasta programa.
2-Rode o comando make, para gerar o executavel do programa.
3-Em seguida rode o comando ./agenda para executar o programa.

#Documentação

1-Abra um terminal na pasta do programa.
2-Rode o comando make docs para gerar a documentação pelo Doxygen.
3-Os arquivos html da documentação estarão na pasta doxygen/html.

#Considerações finais
Não consegui implementar as funçoes de ordenação por inicial, nem por mes. 
Não consegui implementar as funções de exclusão de contato e alteração de contato.
