#Trabalho Ordenação de Pontos Flutuantes

#O programa recebe pela linha de comando dois arquivos, o de entrada (com os valores em forma desordenada) e o de saida (onde os valores estarão ordenados)

#Para a execução ./tda_ordenacao arquivo_entrada arquivo_saida

#Funcionamento

# ->Após abrir corretamente o arquivo_entrada, o programa irá definir a quantidade de valores presentes no arquivo e salvar numa variável;

# ->Irá alocar memória suficiente para criar um vetor de ponteiros capaz de suportar a memória de todos os valores do arquivo;

# ->Passa por uma função que vai salvar cada valor em um ponto do ponteiro de vetor;

# ->Lança o ponteiro do vetor para a função que ordena os valores;

# ->A função ordena os valores e após isso, o vetor ordenado é impresso no arquivo_saida;

#O programa retorna 0 como valor de conclusão.
